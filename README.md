# BNPL UAE Startup Financial Model

## Overview
This repository contains a comprehensive financial model for a Buy Now, Pay Later (BNPL) fintech startup based in the United Arab Emirates. The company offers 3X/4X payment splits, integrated with 210 merchants, targeting young expatriates. Revenue streams include merchant fees, soft late fees, and card interchange.

## Objective
The model supports a **Seed+ fundraising round** of $2.5M, projecting revenue, GMV, and profitability over multiple years. It is designed to help investors understand key drivers, assumptions, and expected returns.

## Features
- Monthly and annual financial projections (2024-2029)
- Revenue, GMV, COGS, OPEX, and EBITDA bridge
- Sensitivity analysis for CAC, growth, and stickiness
- VC exit modeling (Scorecard and VC method)
- LTV, take rate, and payback period calculations done using Power Query
- Scenario analysis for low, base, and high cases

## Assumptions & Drivers
Key assumptions include:
- Average transaction per user, growth rates, stickiness, churn, CAC
- Revenue streams: merchant fees, late fees, interchange
- Operating costs: salaries, G&A, tech & cloud, compliance
- BNPL transactions and cost of funds factors

- ## Notes
- Some calculations such as **LTV, Take Rate, Payback Period** were performed using Power Query to simplify data processing.
- Historical data is used for base assumptions.
